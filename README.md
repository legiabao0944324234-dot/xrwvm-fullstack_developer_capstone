# xrwvm-fullstack_developer_capstone

## Project: Best Cars Dealership Full Stack Application

A full-stack web application built with Django (backend) and React (frontend) for managing car dealership reviews.

### Features
- User authentication (register/login/logout)
- View dealerships by state
- Read and post dealer reviews
- Sentiment analysis on reviews
- Django admin interface

### Tech Stack
- Backend: Django 4.2, Python
- Frontend: React.js
- Database: SQLite (dev), IBM Cloudant (NoSQL for reviews)
- Sentiment Analysis: IBM Watson NLU / Custom microservice
